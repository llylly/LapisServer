# 勘误说明

## Getservice

### 阿里文档勘误

请求参数max-keys取值最大值1000，默认值1000，应当为整数类型，而文档中描述为字符串类型

### 项目文档勘误

请求参数max-keys取值最大值1000，默认值1000，应当为整数类型，而文档中描述为字符串类型
